# Do I Still Think for Myself? Book Website

This folder is a static website for the book. Upload everything in this folder to
any static hosting service, including Netlify, Vercel, GitHub Pages, or a regular
web hosting account.

Main links after publishing:

- Website: `https://book.nehachinnasani.com/`
- Reader section: `https://book.nehachinnasani.com/#reader`
- Direct PDF: `https://book.nehachinnasani.com/assets/do-i-still-think-for-myself.pdf`

Keep the `assets` folder next to `index.html` and `styles.css` so the cover
images and PDF load correctly.
